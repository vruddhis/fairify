What does CST do?

Counterfactual Sentence Testing (CST) deals with sentence augmentation. As the word "counterfactual" suggests, it converts the protected attribute or subject into its counterpart and tests it for model outputs, displaying any variations, in turn.
Example:
The original input - A [girl] must focus on learning to cook.
The counterfactual input - A [boy] must focus on learning to cook.

What does the graph tell you about your model?

The graph shows the aggregate performance which is the probability that the model generates the outputs when given the respective input categories. If one category is more than the other, it shows a possible bias.
